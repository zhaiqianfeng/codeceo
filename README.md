#### 目录含义
```
.
├── cartoon 存放漫画
├── gif 存放动图图片
├── picture 存放静态图片
├── story 存放各种故事、小说
└── video 存放视频
```

#### 文件
- 文件在各对应的目录中存放在日期(yyyyMMdd，如20200601)中，如果目录不存在，请创建；
- 文件命名格式：自然序号-非中文名字.格式，如1-codeceo.gif
- 尽量不要在文件夹名字中出现空格
- 图片请尽量压缩大小, 并 删除 EXIF 信息, 图片质量过低的 Pull Request 会被 Close
